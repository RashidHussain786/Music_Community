# Music Community
## Vedio Calling With Strangers

-  submit your song and check whether any person is present who listen same song at the same time.
- video calling with that person with similar music.
- ✨Magic ✨

## Frontend 

- React Js 
- Material UI 

## Backend
- Express
- Node Js
- Socket.Io
- MySql
